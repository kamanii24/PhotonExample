# PhotonExample

Photon(PUN)の初期実装のサンプルプロジェクトです。  
複数端末でマルチプレイするまでに必要な最低限の処理、ロビー(マスターサーバ)接続 / ルーム作成 / 入室 / 退室を実装しています。  
*※PUN本体はリポジトリに入っていないのでAssetStoreからインポートしてください。*

**Build環境**  
Unity2018.2.3f1  
macOS High Sierra(v10.13.6)
